WordPress 5.0 Beta 3

Posted November 5, 2018 by Gary Pendergast. Filed under Development, Releases.

O WordPress 5.0 Beta 3 já está disponível!

Esta versão ainda está em desenvolvimento, portanto não recomendamos seu uso em ambientes de produção. Considere criar um site de testes para testar a nova versão.

Existem duas formas de testar o WordPress 5.0 beta: com o plugin WordPress Beta Tester (você deve usar “bleeding edge nightlies”) ou você pode baixar a versão aqui (zip).

O lançamento do WordPress 5.0 está previsto para 19 de novembroe precisamos da sua ajuda para que isso aconteça. Aqui estão alguns ajustes importantes que fizemos desde o Beta 2:

Editor de blocos

O editor de blocos foi atualizado para incluir todos os recursos e ajustes da versão 4.2 do Gutenberg. Além disso, existem alguns ajustes e recursos novos, como por exemplo:

  • Suporte ao meta box de "Campos personalizados".
  • Melhoria na confiabilidade das solicitações à API REST.
  • Vários pequenos ajustes e melhorias.

Twenty Nineteen (2019)

O Twenty Nineteen foi atualizado a partir de seu repositório no GitHub. Você pode verificar várias novidades nesta versão:

  • Adicionado suporte à atualização seletiva de widgets no Personalizar.
  • Adicionado suporte à mídias incorporadas responsivas.
  • Ajustes para melhorar a leitura e funcionalidades em dispositivos móveis.
  • Ajustes em blocos aninhados aparecendo mais largos do que deveriam.
  • Correção de erros em versões mais antigas do PHP e no IE11.

Como ajudar

Ajude a tradução em Português! Ajude-nos a traduzir o WordPress para o português!

Se você é fluente em inglês e se sente apto a contribuir com código ou testes, nós temos vários bug scrubs agendados para esta semana e nós adoraríamos envolver o máximo de pessoas possível para termos certeza de que todos os erros informados receberam a devida atenção.

Se acha que encontrou um erro, você pode publicá-lo na área Alpha/Beta do fórum oficial. Nós vamos adorar ouvir você! Se puder escrever um relatório de um erro que outras pessoas podem reproduzir, envie-o para o WordPress Trac (em inglês), onde você também pode encontrar uma lista dos erros já conhecidos.
