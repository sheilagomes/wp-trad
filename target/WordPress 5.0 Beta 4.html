WordPress 5.0 Beta 4

Posted November 13, 2018 by Gary Pendergast. Filed under Development, Releases.

O WordPress 5.0 Beta 4 já está disponível!

Esta versão ainda está em desenvolvimento, portanto não recomendamos seu uso em ambientes de produção. Considere criar um site de testes para testar a nova versão.

Existem duas formas de testar o WordPress 5.0 beta: com o plugin WordPress Beta Tester (você deve usar “bleeding edge nightlies”) ou você pode baixar a versão aqui (zip).

A data de lançamento do WordPress 5.0 mudou. A nova data é 27 de novembro e precisamos da sua ajuda para que isso aconteça. Aqui estão alguns ajustes importantes que fizemos desde o Beta 3:

Editor de blocos

O editor de blocos foi atualizado a partir da versão 4.3 do Gutenberg e as maiores mudanças incluem:

  • Uma API de Anotações, permitindo que plugins incluam dados contextuais enquanto você escreve.
  • Melhoria na consistência da navegação entre blocos feita pelo teclado, assim como ajustes nas diferentes áreas da interface.
  • Melhoria na acessibilidade, com marcações e itens para leitura de tela adicionais.

Além disso, foram corrigidos alguns erros que apareceram na versão beta 3:

  • Melhor suporte a plugins que fazem uso mais avançado das meta boxes.
  • Suporte a concatenação de scripts.
  • Chamadas ajax ocasionalmente causavam erros de PHP.

Internacionalização

Adicionamos uma API para tradução de strings em arquivos JavaScript dos seus plugins e temas! E como o editor de bloco já está usando essa API, você também já pode começar a usá-la. Leia as notas de desenvolvimento para saber mais.

Twenty Nineteen (2019)

O Twenty Nineteen está sendo trabalhado no seu repositório do GitHub. Esta atualização contém vários ajustes, incluindo:

  • Menus agora têm suporte apropriado a interações por toque e teclado.
  • Um menu de rodapé foi adicionado para links de página secundários.
  • Melhoria na compatibilidade com versões anteriores do WordPress.

Temas padrão

Todos os temas padrão mais antigos, do Twenty Ten até o Twenty Seventeen, tiveram seus estilos no editor de blocos melhorados.

Como ajudar

Você lê e escreve bem em inglês? Ajude-nos a traduzir o WordPress para o português! 

Se acha que encontrou um erro, você pode publicá-lo na área Alpha/Beta do fórum oficial. Nós vamos adorar ouvir você! Se puder escrever um relatório de um erro que outras pessoas podem reproduzir, envie-o para o WordPress Trac (em inglês), onde você também pode encontrar uma lista dos erros já conhecidos.
