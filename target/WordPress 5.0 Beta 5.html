WordPress 5.0 Beta 5

Posted November 16, 2018 by Jonathan Desrosiers. Filed under Development, Releases.

O WordPress 5.0 Beta 5 já está disponível!

Esta versão ainda está em desenvolvimento, portanto não recomendamos seu uso em ambientes de produção. Considere criar um site de testes para testar a nova versão.

Existem duas formas de testar o WordPress 5.0 Beta: com o plugin WordPress Beta Tester (você deve usar “bleeding edge nightlies”) ou você pode baixar a versão aqui (zip).

Lembrete: A data de lançamento do WordPress 5.0 mudou. A nova previsão de lançamento é 27 de novembro e precisamos da sua ajuda para que isso aconteça. Aqui estão alguns ajustes importantes que fizemos desde o Beta 4:

Editor de blocos

O editor de blocos foi atualizado a partir da versão 4.4 do Gutenberg e as maiores mudanças incluem:

  •  Um painel de links permanentes foi incluído na barra lateral de documento e agora é mais fácil encontrá-la.
  • Os painéis de documento do editor agora podem ser removidos programaticamente.
  • O indicador de envio de imagens e galerias foi substituído por uma imagem que fica girando e depois desaparece.
  • Os blocos de edição de texto e código agora ocupam toda a largura do editor.
  • A manipulação de imagens foi melhorada. As imagens agora ocupam a quantidade de espaço certa para temas com editores mais largos , como o Twenty Nineteen.
  • Estilos de hover agora são devidamente desabilitados em dispositivos móveis.
  • O módulo de i18n (internacionalização) foi refatorado, recebendo ganhos significativos de desempenho.

Além disso, alguns errinhos chatos foram corrigidos:

  • Melhor tratamento de links sem um attributo href, que apareciam como undefined.
  • Texto em japonês (caracteres com double byte) agora estão disponíveis na lista de blocos.
  • Melhor tratamento de encodings diferentes de texto, como emojis por exemplo, na validação do bloco.

A lista completa de mudanças está no post de lançamento do Gutenberg 4.4.

Suporte para PHP 7.3

Os últimos erros conhecidos de compatibilidade com o PHP 7.3 foram corrigidos. Você pode saber mais sobre o PHP 7.3 e o WordPress lendo as notas de desenvolvimento no blog Make WordPress Core.

Twenty Nineteen (2019)

O esforço para que o Twenty Nineteen esteja pronto na época de lançamento da versão 5.0 continha no repositório do GitHub. Esta atualização contém vários ajustes, incluindo:

  • Suporte para a class .button.
  • Correção de font-weights para títulos.
  • Melhoria no suporte para barra de ferramentas fixas no editor.
  • Melhoria no contraste e legibilidade das cores personalizadas usadas durante a seleção de texto
  • Correção do editor para que a área de meta boxes do Gutenberg não se sobreponha ao conteúdo

Como ajudar

Você lê e escreve bem em inglês? Ajude-nos a traduzir o WordPress para o português! 

Se acha que encontrou um erro, você pode publicá-lo na área Alpha/Beta do fórum oficial. Nós vamos adorar ouvir você! Se puder escrever um relatório de um erro que outras pessoas podem reproduzir, envie-o para o WordPress Trac (em inglês), onde você também pode encontrar uma lista dos erros já conhecidos.
