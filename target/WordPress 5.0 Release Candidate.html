WordPress 5.0 Release Candidate

Posted November 23, 2018 by Matias Ventura. Filed under Development, Releases.

O primeiro <i>release candidate</i> (versão candidata, em tradução livre) do WordPress 5.0 já está disponível!

Isso é bem importante, significa que estamos cada vez mais próximos do WordPress 5.0. A data de lançamento do WordPress 5.0 foi adiada e não será mais no dia 27, para que o RC seja completamente testado. Uma data de lançamento final será anunciada em breve, baseada nos comentários sobre o RC. Este é um grande lançamento e precisa da sua ajuda. Se você ainda não testou a versão 5.0, a hora é essa! 

Para testar o WordPress 5.0 beta, você pode usar o plugin WordPress Beta Tester ou você pode baixar o release candidate aqui (zip).

Quais são as novidades do WordPress 5.0?

O WordPress 5.0 traz o novo editor de posts baseado em blocos. O primeiro passo para um futuro emocionante, com uma experiência de edição mais simples em todas as partes do seu site. Você terá mais flexibilidade nas formas como o conteúdo é exibido, esteja você construindo o seu primeiro site, atualizando o seu blog ou seja você programador.

O editor de blocos está sendo usado em mais de um milhão de sites e nós acreditamos que ele está pronto para ser usado em todos os sites WordPress. Mesmo assim, entendemos que alguns sites talvez precisem de mais tempo. Se é o seu caso, instale o plugin Editor Clássico e você vai continuar usando o editor de posts clássico quando atualizar seu site para o WordPress 5.0.

O Twenty Nineteen (2019) é o novo tema padrão do WordPress. Ele traz estilos personalizados para os blocos disponíveis por padrão no WP 5.0. O Twenty Nineteen foi pensado de forma que você possa usá-lo em vários tipos de sites diferentes. Se você tem um blog de fotografia, um site para seu novo negócio ou ajudando uma ONG, o Twenty Nineteen é flexível o suficiente para atender às suas necessidades.

O editor de blocos é uma grande mudança, mas isso não é tudo. Também fizemos algumas mudanças menores, incluindo:

  • Todos os temas padrão anteriores, do Twenty Ten (2010) até o Twenty Seventeen (2017), foram atualizados para dar suporte ao editor de blocos.
  • Você pode melhorar a acessibilidade do conteúdo que você escreve, agora que rótulos ARIA simples podem ser salvos em posts e páginas.
  • O WordPress 5.0 tem oficialmente suporte para versão 7.3 do PHP: se você está usando uma versão anterior, sugerimos que você atualize o PHP do seu site.
  • Desenvolvedores agora podem adicionar strings traduzíveis diretamente em seus códigos JavaScript, usando os novos pacotes de idioma JavaScript.

Você pode saber mais sobre os ajustes e mudanças desde o Beta 5 no post de atualização mais recente.

Para mais detalhes sobre as novidades da versão 50, leia também os posts sobre Beta 1, Beta 2, Beta 3, Beta 4 e Beta 5.

Desenvolvedores de plugins e temas

Teste seus plugins e temas com o WordPress 5.0 e atualize para 5.0 a versão no campo Tested up to do arquivo readme. Se encontrar problemas de compatibilidade, tenha certeza de publicar no fórum para que possamos agir antes do lançamento final. Um guia aprofundado com foco nas mudanças para desenvolvedores será publicado em breve no blog de desenvolvimento dos arquivos básicos. Enquanto isso, você pode rever as notas de desenvolvimento da versão 5.0.

Como ajudar

Você lê e escreve bem em inglês? Ajude-nos a traduzir o WordPress para o português! 

Se acha que encontrou um erro, você pode publicá-lo na área Alpha/Beta do fórum oficial. Nós vamos adorar ouvir você! Se puder escrever um relatório de um erro que outras pessoas podem reproduzir, envie-o para o WordPress Trac (em inglês), onde você também pode encontrar uma lista dos erros já conhecidos.
