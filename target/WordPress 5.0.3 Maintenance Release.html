WordPress 5.0.3 – Atualização de manutenção

Posted January 9, 2019 by Jonathan Desrosiers. Filed under Releases.

O WordPress 5.0.3 já está disponível!

A versão 5.0.3 é uma atualização de manutenção que corrige 36 erros e inclui 7 atualizações de desempenho. O foco deste lançamento foram ajustes no editor de blocos e correções de erros significativos e regressões.

Conheça alguns dos destaques:

  • Quinze correções de erros e melhorias relacionados ao editor de blocos foram adicionaos aos temas padrão. Tenha certeza de atualizá-los para uma melhor experiência na edição com blocos.
  • Dois erros relacionados à internacionalização (I18N) do editor de blocos foram corrigidos
  • Usuários com JavaScript desabilitado agora recebem um aviso ao tentar usar o editor de blocos.
  • Alguns errors de PHP no Personalizar foram corrigidos.
  • Alguns problemas ao enviar arquivos com tipos comuns como CSVs, foram corrigidos.

Para uma lista completa das mudanças, veja a lista de tickets no Trac o registro de alterações ou leia um resumo mais técnico no blog Make WordPress Core (em inglês).

Você pode baixar o WordPress 5.0.3 ou ir em Painel → Atualizações e clicar em Atualizar agora. Sites com suporte a atualizações automáticas em segundo plano já começaram a se atualizar.
